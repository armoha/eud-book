
#  Minimap State
Address   | 6D5BBE
----------|-------------
Player ID | 0 (Byte Offset: 0)
Size 	  | 4
Length 	  | 1
SC:R      | 

Minimap State -
Pressing Tab, Shift+tab changes this 0x00 - default
0x01 - Enemies red, allies yellow
0x02 - Enemies red, allies yellow, ememy units red, ally units yellow, your units teal

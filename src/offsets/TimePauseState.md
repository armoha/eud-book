#  Time Pause State
Address   | 58F04C
----------|-------------
Player ID | 4922 (Byte Offset: 0)
Size 	  | 4
Length 	  | 1
SC:R      | Simple Data

If FALSE, the value of the address is 0, and the countdown timer is not paused.
if TRUE, the value of the address is greater than 0, and the countdown timer is paused.

#  Game Speed
Address   | 6CDFD4
----------|-------------
Player ID | 331548 (Byte Offset: 0)
Size 	  | 4
Length 	  | 1
SC:R      | Backed By Code

The current game speed. The values are as follows:

0 = Slowest
1 = Slower
2 = Slow
3 = Normal
4 = Fast
5 = Faster
6 = Fastest

Note:
SC:R no longer uses this address. The value is always 0 in SC:R.
